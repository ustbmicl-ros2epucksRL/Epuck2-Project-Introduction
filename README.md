# Epuck2-Project-Introduction
本项目主要研究内容为基于ROS2实现epuck2群智机器人基础功能及在各类特定场景中发挥多机器人优势，已参加相关比赛srtp国家级项目，已获中国大学生计算机设计大赛三等奖。
